# Intelligent-car
这是我的毕业设计，有兴趣可以参考着自己搭建，比树莓派什么的有难度多了，反正电路代码图纸全部都有，就是个遥控的智能车，能图传，能做点图像处理
视频介绍http://www.bilibili.com/video/av4730503/
# 主要的内容
CPU：STM32
上位机：C#
图像处理：OpenCv
C#写的软件，智能车用STM32控制，ESP8266模块连接电脑软件，软件用TCPserver建立端口监听，实现转向控制，图传，图像处理，传感器数据交互。
